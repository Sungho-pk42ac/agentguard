# Security Policy

AgentGuard is a security-focused project, but it is still an early MVP. Please report security issues responsibly.

## Supported versions

| Version | Supported |
|---|---|
| `0.1.x` | Yes |

## Reporting a vulnerability

Please do **not** open a public issue for vulnerabilities involving bypasses, unsafe disclosure, or real credentials.

Instead, use GitHub's private vulnerability reporting if enabled on the repository, or contact the maintainer through the GitHub profile.

Include:

- affected version or commit
- reproduction steps
- expected behavior
- actual behavior
- impact assessment
- whether any real secret or private data was involved

## Handling test data

Never submit real secrets, tokens, customer data, private logs, or proprietary agent transcripts. Use redacted or synthetic examples only.
