# AX 데모 시나리오 매트릭스

이 문서는 AX 해커톤 심사위원이 낯선 기업 문제를 30초 안에 AgentGuard 데모 팩으로 연결해 볼 수 있도록 만든 한국어 우선 매트릭스입니다. AgentGuard의 현재 범위는 **PR diff + MCP + transcript evidence**입니다. OWASP, MCP, agent component scan framing은 공통 언어로만 빌려 쓰며, 인증·준수·도입 실적을 주장하지 않습니다.

## 사용 방법

1. 저장소 루트에서 실행한다고 가정합니다. 예: `/opt/data/repos/agentguard` 또는 clone한 repo root.
2. 심사위원이 제시한 기업 문제와 가장 가까운 행을 고릅니다.
3. `Company problem`을 한 문장으로 읽고, 바로 `30-second demo angle`을 보여줍니다.
4. 같은 행의 `AgentGuard evidence` 명령을 실행해 초기 `BLOCK` 또는 `REVIEW`를 보여줍니다.
5. `Fix/policy/PASS story`로 어떤 수정과 정책 조건을 충족하면 승인 가능한 `PASS` 또는 검토 가능한 상태가 되는지 설명합니다.

## 발표 가드레일

- AgentGuard는 현재 `agentguard scan-diff`, `agentguard scan-mcp`, `agentguard scan-log`로 데모 팩의 PR diff, MCP config, agent transcript/log를 검사합니다.
- `scan-mcp`는 Claude Desktop(`claude_desktop_config.json`)과 Cursor(`.cursor/mcp.json`) 형식의 로컬 MCP client config도 같은 방식으로 검사하며, `scan-log`는 plain text log 외에 Codex/Hermes 스타일 JSONL agent transcript도 이해합니다.
- CLI commands, rule IDs, JSON, SARIF, machine-facing terms는 English-compatible로 유지합니다.
- 이 매트릭스는 OWASP LLM Top 10 또는 MCP 표준 준수를 주장하지 않습니다. 공통 위험 어휘와 권한/credential 노출 프레임만 참고합니다.
- 모든 fixture는 합성 데이터이며 실제 개인, 조직, 로고, 토큰, 배포 실적을 포함하지 않습니다.
