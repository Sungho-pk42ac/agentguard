# Roadmap

AgentGuard's roadmap is intentionally small-slice. Each item should be shippable with tests, examples, and docs.

## Good first issues

- Add more synthetic transcript fixtures for approval-required operations.

## Near-term

- Expanded MCP permission graph.
- Safer default policy examples for common AI-agent workspaces.

## Later

- Dashboard for agent audit trails.
- Configurable severity overrides.
- Integration recipes for additional CI providers.

## Non-goals

- Runtime sandboxing.
- Credential storage.
- Network-based scanning.
- Replacement for full security review.
